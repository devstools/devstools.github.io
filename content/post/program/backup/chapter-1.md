## OOP

- Object-oriented programing is a way to develop software application using real-world terminologies to create entities that interact with one another using objects.

4 Componenet of object-oriented:

- Class: define the structure using methods and properties to describle real-world entity

- Methods: A method represents a particular behavior, this is action of real-world entity example
Human class can have method: talk, speak, doTask...

- Properties: properties hold the data temporarily during the execution of an application

- Interface: is a design of class in software development. is helpul to make class can inheritance and polymorphism to make application more flexible.

4 Properties of OOP

1 - Abstraction: I

2 - Encapsulation

3 - Inheritance

4 - Polymorphism
